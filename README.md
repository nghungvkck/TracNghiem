# TracNghiem Static Site

Project nay da duoc to chuc lai de chay nhu mot trang web tinh.

## Cau truc

- `index.html`: trang chinh de deploy
- `assets/`: CSS, JS, anh va tai nguyen can thiet
- `Bài trắc nghiệm tổng hợp_ Attempt review.html`: ban goc de doi chieu neu can

## Cach deploy len GitHub Pages

1. Day toan bo thu muc nay len repo GitHub.
2. Vao `Settings` -> `Pages`.
3. Chon source la branch chinh va thu muc `/ (root)`.
4. Mo link GitHub Pages, trang se load tu `index.html`.

## Luu y

- Ban `index.html` da duoc chuyen sang kieu static, khong con phu thuoc PHP backend.
- Mot so link Moodle noi bo da duoc vo hieu hoa vi khong the dung tren host tinh.
